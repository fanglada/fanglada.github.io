Website Jovent del Bruel

This project was created with the goal of promoting the Jovent del Bruel association and providing a tool to sell and validate QR tickets for their activities.

It is developed using Astro, Tailwind, and MySQL, and integrates Redsys for QR ticket purchase transactions. The QR codes can be scanned to verify their authenticity and check if they have already been used, ensuring security and control for the event organizers.

Sale with control of stock

This system is designed to track inventory and orders in bar services, allowing users to generate tickets for each sale. These tickets (orders) are sent to the bar for preparation. The system is scalable, enabling easy product updates and automatically generating tickets without any limits.

It is developed using web technologies such as HTML, PHP, CSS with Tailwind, and JavaScript. A MySQL database is used to store all products and their stock. The system generates tickets for each sold item, which are sent via an API to be printed. This comprehensive solution is unified within a Raspberry Pi, providing a compact and efficient platform for managing orders and inventory.
